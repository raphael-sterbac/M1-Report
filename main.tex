This formulation is interesting in that it embeds all the rules of cumulative universe hierarchies
in functoriality and naturality conditions. One can give a more common presentation with typing rules,
that corresponds to unfolding the diagrams of the previous definition.

\begin{proposition}\label{nat-hierarchy-rules}
    A natural universe hierarchy is equivalently defined by the more common presentation.
    
    \begin{mathpar}
        \ebrule[dfun-decoding]{
            \hypo{\Gamma\vdash a: \Univ_n}
            \hypo{\Gamma,x:\El_n(a)\vdash b: \Univ_n}
            \infer2{\Gamma\vdash \El_n(\Pi^n \  a \  b) \equiv \Pi_{x: \El_n(a)} \El_n(b \brk{x})}
        }
        \and
        \ebrule[dfun-lifting]{
            \hypo{\Gamma\vdash a: \Univ_n}
            \hypo{\Gamma,x:\El_n(a)\vdash b: \Univ_n}
            \hypo{\Gamma\vdash n\leq p}
            \infer3{\Gamma\vdash \; \uparrow_n^p(\Pi^n \  a \  b) \equiv\; \Pi^p \  \uparrow_n^p(a) \  \uparrow_n^p(b)}
        }
        \and
        \ebrule[univ-code-decoding]{
            \hypo{}
            \infer1{\Gamma\vdash \El_n(\Univ_m^n) \equiv \Univ_m}
        }
        \and
        \ebrule[univ-code-lifting]{
            \hypo{\Gamma\vdash m < n \leq p}
            \infer1{\Gamma\vdash \; \uparrow_n^p(\Univ_m^n) \equiv \Univ_m^p : \Univ_p}
        }
        \and
        \ebrule[lifting-decoding]{
            \hypo{\Gamma\vdash a : \Univ_m}
            \hypo{\Gamma\vdash m \leq n}
            \infer2{\Gamma\vdash \El_n(\uparrow_m^n(a)) \equiv \El_m(a)}
        }
        \and
        \ebrule[lifting-lifting]{
            \hypo{\Gamma\vdash a : \Univ_m}
            \hypo{\Gamma\vdash m\leq n\leq p}
            \infer2{\Gamma\vdash \; \uparrow_n^p(\uparrow_m^n(a)) \equiv\; \uparrow_m^p(a): \Univ_p}
        }
    \end{mathpar}
\end{proposition}

\section{Normalisation}

Let $\mathcal{T}$ be the the dependent type theory with a natural universe hierarchy just like
in the previous section.  We are going to prove that normalisation holds for that theory $\mathcal{T}$ using the new technique of \cite{sconing}.
An higher-order model of $\mathcal{T}$ consists of the following data: 

\begin{align*}
    \Ty &: \Set,\\
    \Tm &: \Ty \rightarrow \Set,\\
    \Pi &: (A:\Ty) \rightarrow (\Tm(A) \rightarrow \Ty) \rightarrow \Ty\\
    \app &: (A:\Ty) \rightarrow (B:\Ty) \rightarrow (f:\Tm(\Pi(A,B))) \rightarrow (a:\Tm(A)) \rightarrow \Tm(B(a)),\\
    \lam &: (A:\Ty) \rightarrow (B:\Ty) \rightarrow (b: (a:\Tm(A)) \rightarrow \Tm(B(a))) \rightarrow \Tm(\Pi(A,B)),\\
    \U_i &: \Ty, && (\forall i \in \bL)\\
    \El_i &: \Tm(\U_i) \rightarrow \Ty, && (\forall i \in \bL)\\
    \uparrow_i^j &: \Tm(\U_i) \rightarrow \Tm(\U_j), && (\forall i \leq j \in \bL)\\
    u_i^j &: \Tm(\U_j), && (\forall i < j \in \bL)\\
    \Pi^i &: (a:\U_i) \rightarrow (\Tm(\El_i(a)) \rightarrow \U_i) \rightarrow \U_i. && (\forall i \in \bL)
\end{align*}

With $\beta$ and $\eta$ rules, and all the functoriality and naturality rules inferred from the previous section.
We write $\cS$ for the syntax of $\cT$, that is the initial first-order model of $\cT$. The constructions of \cite{sconing} extend
to $\cS$, and we notably get the relative induction principle for the functor $F : \mathsf{Ren}_\cS \rightarrow \cS$, from the initial first-order
renaming algebra $\mathsf{Ren}_\cS$.


Neutral and normal forms are defined internally to $\Psh(\mathsf{Ren}_\cS)$ as 
\[\Tm_N, \Neut : (A: \cS_F.\Ty(1_{\cS_F})) \rightarrow (a: \cS_F.\Tm(1_{\cS_F},A)) \rightarrow \Set\]
and
\[\Ty_N : (A: \cS_F.\Ty(1_{\cS_F})) \rightarrow \Set\]
and are generated by:

    \begin{align*}
        \Pi^\nf &: \Ty_N(A) \rightarrow \Ty_N(B) \rightarrow \Ty_N(\Pi(A,B))\\
        \var^\neut &: (x:\Var(A)) \rightarrow \Neut(A,\var(x)).\\ 
        \app^\neut &: \Ty_N(A) \rightarrow  \Ty_N(B) \rightarrow \Neut(\Pi(A,B),f) \rightarrow \Ty_N(A,a) \rightarrow \Neut(B[a],\app(A,B,f,a))\\
        \lam^\nf &: (a:\Var(A)) \rightarrow \Tm_N(B[a],b[a]) \rightarrow \Tm_N(\Pi(A,B),\lam(b))\\
        \U_i^\nf &: \Ty_N(U_i) && (\forall i \in \bL)\\
        \El_i^\nf &: \Neut(U_i,a) \rightarrow \Ty_N(El_i(a)) && (\forall i \in \bL)\\
        (\uparrow_j^i)^\nf &: \Neut(\U_j,a) \rightarrow \Tm_N(\U_i,a) && (\forall i < j \in \bL)\\
        (\U_j^i)^\nf &: \Tm_N(\U_i,\U_j^i) && (\forall i<j \in \bL)\\
        (\Pi^i)^\nf &: \Tm_N(\U_i,a) \rightarrow (\Tm_N(\El_i(a),x) \rightarrow \Tm_N(\U_i,b[x])) \rightarrow \Tm_N(\U_i, \Pi^i(a,b)) && (\forall i \in \bL)\\
        \mathsf{CodeNe}^\nf_i &: \Neut(\U_i,a) \rightarrow \Tm_N(\U_i,a) && (\forall i \in \bL)\\
        \mathsf{DecodeNe}^\nf_i &: \Neut(\U_i,a) \rightarrow \Neut(\El_i(a),b) \rightarrow \Tm_N(\El_i(a),b) && (\forall i \in \bL)
    \end{align*}

The goal of normalisation is now to prove that every term and type has a unique normal form, that is we have two functions:
\[ \forall(A:\cS_F.\Ty(1_{\cS_F}) \rightarrow \text{isContr}(\Ty_N(A))\]
\[ \forall(A:\cS_F.\Ty(1_{\cS_F}) \ (a:\cS_F.\Tm(1_{\cS_F},A)) \rightarrow \text{isContr}(\Tm_N(A,a))\]

Those are obtained by applying the relative induction principle of \cite{sconing}, over the \textit{normalisation} displayed higher-order model $\mathbb{S}^\bullet$ over $\mathsf{Tele}\mathbb{S}$,
that we now define.

A displayed type $A^\bullet : \Ty^\bullet(A)$ over $A : \cS_F.\Ty(1_{\cS_F})$ is a quadruple $(A_p^\bullet,A_N^\bullet,A_u^\bullet,A_q^\bullet)$ consisting of:
\begin{itemize}
    \item a \textit{logical predicate}  $A_p^\bullet : \cS_F.\Tm(1_{\cS_F},A) \rightarrow \Set$,
    \item a \textit{normal form} $A_N^\bullet : \Ty_N(A)$,
    \item an \textit{unquoting} function $A_u^\bullet : (a: \cS_F.\Tm(1_{\cS_F},A)) \rightarrow \Neut(A,a) \rightarrow A_p^\bullet(a)$,
    \item a \textit{quoting} function $A_q^\bullet : (a: \cS_F.\Tm(1_{\cS_F},A)) \rightarrow A_p^\bullet \rightarrow \Tm_N(A,a)$.
\end{itemize}
  

A displayed term $a^\bullet : A^\bullet$ over $a : \cS_F.\Tm(1_{\cS_F},A)$ is an element of $A_p^\bullet(a)$.
The logical predicate and constructions for $\Pi$, $\app$, $\lam$ and $\var$ are defined as in \cite{sconing}. Here, we only emphasize the relevant
constructions for universes.\\

        In the following we will write :
        \[
            \mathsf{Lift}_i^j(k^\neut) := 
            \left\{\begin{array}{lr}
                    \mathsf{CodeNe}^\nf_i(k) & \text{if } i=j \\
            (\uparrow_i^j)^\nf(k^\neut) & \text{if } i \neq j
            \end{array}\right. 
        \] 

        so that the identity lifts are well-handled. We define $\U_i^\bullet$ as the following :
            \begin{itemize}
                \item $(\U_i)_p^\bullet$ where $(\U_i)_p\; a$ is the set of $5$-tuples $(a_p,a_N,a_T,a_l,a_u,a_q)$ such that :
                    \begin{align*}
                        a_p &: \cS_F.\Tm(1_{\cS_F},\El_i(a)) \rightarrow \mathcal{U}_i && \text{(logical predicate)}\\
                        a_N &: \Tm_N(\U_i,a) && \text{(normal form)}\\
                        a_T &: \Ty_N(\El_i(a)) && \text{(normal form for the decoding)}\\
                        a_l &: (j \geq i : \bL) \rightarrow \Tm_N(\U_j,\uparrow_i^j(a)) && \text{(normal form for the lift)}\\
                        a_u &: (k: \cS_F.\Tm(1_{\cS_F},\El_i(a))) \rightarrow \Neut(\El_i(a),k) \rightarrow a_p(k)&& \text{(unquoting function)}\\ 
                        a_q &: (u: \cS_F.\Tm(1_{\cS_F},\El_i(a))) \rightarrow a_p \rightarrow \Tm_N(\El_i(a),u) && \text{(quoting function)} 
                    \end{align*}

                \item $(\U_i)_N^\bullet = \U_i^\nf$.

                \item $(\U_i)_u^\bullet \;  k^\neut = (\lambda u \mapsto \Neut(\El_i(k),u),\ \mathsf{CodeNe}^\nf_i(k), \ \El_i^\nf(k^\neut), \ \lambda m \mapsto \mathsf{Lift}_i^m(k), \ \lambda \, k' \mapsto k',\ \lambda \, u^\bullet \mapsto \mathsf{DecodeNe}^\nf_i (k^\neut, u^\bullet))$ 

                \item $(\U_i)_q^\bullet \; a^\bullet = a_N$.

            \end{itemize}

            We also define :
            \begin{align*}
                \El_i^\bullet(a^\bullet) &:= (a_p,\ a_T,\ a_u,\ a_q)\\
                (\uparrow_i^j)^\bullet(a^\bullet) &:= (a_p,\ a_l(j),\ a_T,\ a_l,\ a_u,\ a_q)\\
                (\U_i^j)^\bullet &:= (\lambda u \mapsto \Tm_N(\U_i,u),\ (\U_i^j)^\nf,\ \U_i^\nf,\ \lambda m \mapsto \mathsf{Lift}_i^m(k)  \ (\U_i)_u,\ \lambda \,u^\bullet  \mapsto u^\bullet )
            \end{align*}

        We now define $(\Pi^i)^\bullet(a^\bullet,b^\bullet)$ as:
            \begin{align*}
                (\Pi^i(a^\bullet,b^\bullet))_p(f) &= (t: \cS_F.\Tm(1_{\cS_F},\El_i(a))) \rightarrow (t^\bullet : a_p)  \rightarrow b_p (t^\bullet,\app(f,t))\\
                (\Pi^i(a^\bullet,b^\bullet))_N &= (\Pi^i)^\nf (a_N, b_N)\\
                (\Pi^i(a^\bullet,b^\bullet))_T &= \Pi^\nf \; a_T \; b_T\\
                (\Pi^i(a^\bullet,b^\bullet))_l &= \Pi^\nf \; a_l \; b_l\\
                (\Pi^i(a^\bullet,b^\bullet))_u (f^\neut) &= \lambda t^\bullet \mapsto b_u(t^\bullet, \app^\neut(f^\neut,a_q(t^\bullet)))\\
                (\Pi^i(a^\bullet,b^\bullet))_q (f^\bullet) &= \lam^\nf(\lambda t \mapsto \textsf{ let } t^\bullet = a_u(\var^\neut(a)) \textsf{ in } b_q(t^\bullet,f^\bullet(t^\bullet)))
            \end{align*}

            Then by the relative induction principle, we obtain a section $\llbracket . \rrbracket$ of $\mathsf{Scone}_{\mathbb{S}^\bullet}$ and we can define the normalisation functions as follows :
            \begin{align*}
                \mathsf{norm}_\Ty(A) &:=  \llbracket A \rrbracket_N\\
                \mathsf{norm}_\Tm(A,a) &:=  \llbracket A \rrbracket_q(\llbracket a \rrbracket).
            \end{align*}

            Then, we can prove by mutual induction on neutrals and normal forms that normalisation is stable, and thus unique following the proof of \cite{sconing}.


\section{A new algebraic presentation for universe hierarchies}

\subsection{The syntax} \label{rules}

In this section we define a new syntax for universe hierarchies, which provides an
algebraic presentation for explicit (open) universes, and can be understood as a
variant of universes à la Coquand.  The main idea is to introduce a "smallness"
judgement, which correspond to the reflection of $\Univ_i$ in the $\Type$ world.
The rules of this system $\mathcal{T}_F$ are described in the following.
\begin{mathpar}
  \ebrule[univ-decoding-small]{
    \hypo{\Gamma\vdash A : \Univ_i}
    \infer1{\Gamma\vdash \decode{A}_i \; \Smll_i}
  }
  \and
  \ebrule[univ-coding-formation]{
    \hypo{\Gamma\vdash A : \Smll_i}
    \infer1{\Gamma\vdash \code{A}_i : \Univ_i}
  }
  \and
  \ebrule[univ-decoding-formation]{
    \hypo{\Gamma\vdash A : \Univ_i}
    \infer1{\Gamma\vdash \decode{A}_i \; \Type}
  }
  \and
  \ebrule[univ-coding-congruence]{
    \hypo{\Gamma\vdash A \equiv B}
    \hypo{\Gamma\vdash A \; \Smll_i}
    \infer2{\Gamma\vdash \code{A}_i \equiv \code{B}_i : \Univ_i}
  }
  \and
  \ebrule[univ-decoding-congruence]{
    \hypo{\Gamma\vdash A \equiv B : U_i}
    \infer1{\Gamma\vdash \decode{A}_i \equiv \decode{B}_i}
  }
\end{mathpar}

In this formulation, we require the rule \textsc{univ-decoding-inj} explicitely, but we don't ask for the closure under type connectives. The other new rule \textsc{univ-decoding-image} states that $\Smll_i$ is the image of $\decode{.}_i$, by requiring explicitely that $\code{.}_i$ is the right-inverse of $\decode{.}_i$.

Note that there is no coherence issue related to this, as we only ask $\code{.}_i$ and $\decode{.}_i$ to be inverses of each other on the $\Smll_i$ part of $\Type$.

\begin{mathpar}
  \ebrule[univ-decoding-inj]{
    \hypo{\Gamma\vdash \decode{A}_i \equiv \decode{B}_i}

  }
\end{mathpar}

\end{mathpar}

        \item $\Pi^i \; a \; b := \code{ \Pi_{x:\decode{a}_i}\; \decode{b}_i\brk{x}}_i$
        \item $\Univ_i^j := \code{\Univ_i}_j$
    \end{itemize}

    This means that the rules of proposition \ref{nat-hierarchy-rules} are directly derivable.
    
    \end{proposition}


\subsection{Equivalence with a strict Tarski-style hierarchy}

Conversely, in a natural universe hierarchy $\cT_T$ like in proposition \ref{nat-hierarchy-rules},
we can show the injectivity of the decoding map. This result is not immediate and relies
on normalisation.

\begin{theoreme}[Admissibility of Injectivity]\label{InjectivityTarki}
    The following rule is admissible in $\cT_T$: 

    \begin{mathpar}
      \ebrule[univ-decoding-inj]{
        \hypo{\Gamma\vdash \El_i(a) \equiv \El_i(b)}
        \infer1{\Gamma\vdash A \equiv B : U_i}
      }
    \end{mathpar}
\end{theoreme}

\begin{proof}
    We can show the normalisation of the theory with an Artin glueing argument. This provides
    a normalisation by evaluation map $\nf : \Type \rightarrow \Type_N$  which is a section
    of the quotient map $\langle . \rangle : \Type_N \rightarrow \Type$, where $\Type_N$ is 
    the sort of normal types. Those are of the form \[M,N = \U_i \; | \; \Pi(M,N) \; | \; \El_i(R)\] where
    $R$ is neutral.
    Then, we proceed by induction on the normal forms $a_N = \nf(a)$ and $b_N = \nf(b)$. Notice that we need to prove a stronger statement : if $\El_i(a) = \El_j(b)$ then $i = j$ and $a = b$.\\

    \underline{First case:} $a_N = \U_j^i$ and $b_N = \U_k^i$. Then, with the equation $\El_i(a_N) = \El_i(b_N)$ we get $\U_k = \U_i$ which implies $i = j$, and we are done : $\U_j^i = \U_k^i$\\

    \underline{Second case:} $a_N = \Pi^i \; c \;d$ and $b_N = \Pi^i \; c' \; d'$. Then the equation $\El_i(a_N) = \El_i(b_N)$ gives us $\Pi \; \El_i(c) \; \El_i(d) = \Pi \; \El_i(c') \; \El_i(d')$.
    From this we can deduce $\El_i(c) = \El_i(c')$ and $\El_i(d) = \El_i(d')$. By the induction hypothesis we get $c = c'$ and $d = d'$, from which we conclude : $\Pi^i \; c \; d = \Pi^i \; c' \; d'$.\\

    \underline{Third case:} $a_N = \uparrow_j^i c$ and $b_N = \uparrow_k^i c'$. Then the equation $\El_i(a_N) = \El_i(b_N)$ gives us $\El_j(c) = \El_k(c')$. By the induction hypothesis we get $j = k$ and
    $c = c'$.


\end{proof}

Hence, when giving models to the theory $\cT_T$ we can interpret the decoding map as a mono $\El_i : \U_i \rightarrowtail \Type$.
Let $\cC$ denote the syntactic CwF of $\cT_T$, then judgements of this theory are presheaves $\Psh(\cC)$, which models an \textit{extensional}
type theory. This will allow us to reason about definitional equality internally, and to derive a new judgement $\Small_i$ as such a presheaf.

    $\cT_F$.\\

\end{proposition}

\subsection{A study of the different problems related to universes}

When trying to build \textit{natural} universe hierarchies, one may encounter different
problems.

\begin{definition}[Problems related to universes]
    We define the following problems.

    \begin{itemize}
        \item\ [StrictEl]: The commutation of decoding with codes for connectives.
        \item\ [StrictLift]: The commutation of lifting coercions with codes for connectives.
        \item\ [InjEl]: The injectivity of decoding. 
        \item\ [InjLift]: The injectivity of lifting.
    \end{itemize}
\end{definition}

Some of the connections between those problems are already well known, and the main
solution paths are listed in the following proposition.

\begin{proposition} We have the following implications.
    \begin{itemize}
        \item (for free) $\underset{\text{Schulman's local universes}}{\Longrightarrow}$ [StrictEl]

        \item\ [InjLift] $\underset{\text{Top universe}}{\Longrightarrow}$ [InjEl]
        \item\ [StrictLift] $\underset{\text{Top universe}}{\Longrightarrow}$ [StrictEl]
        \item\ [InjLift] $\underset{\text{Realignment}}{\Longrightarrow}$ [StrictLift]
    \end{itemize}
\end{proposition}

What we showed in the previous section leads to less known implications:


\begin{proposition} We proved the following.
    \begin{itemize}
        \item\ [InjEl] $\underset{\text{Syntax}}{\Longrightarrow}$ [InjLift]
        \item\ [StrictLift, StrictEl] $\underset{\text{Gluing}}{\Longrightarrow}$ [InjEl]
        \item\ [StrictLift, StrictEl] $\underset{\text{Section } 2.2}{\Longrightarrow}$ [Fuss-Free]
    \end{itemize}

    Additionally:

    \begin{itemize}
        \item\ [InjEl, StrictEl] $\underset{\text{Syntax}}{\Longrightarrow}$ [StrictLift]
    \end{itemize}

    Thus : 
    \[ \text{[StrictLift,StrictEl]} \Longleftrightarrow \text{[InjEl,StrictEl]} \Longleftrightarrow \text{Fuss-Free}\]
\end{proposition}

This equivalence would solve a practical problem if we had a natural way to get an injective hierarchy
in an arbitrary $\infty$-topos model, or we had a natural way to get a strict hierarchy in an
arbitrary $\infty$-topos model. In either case, we could use the results above to interpret
a very strong type theory.\\

We advocate for injectivity, that we believe to be the easier property in semantics. Hence, our
result may allow the construction of new models which would not necessarily use properties
such as realignment to get strict lifts and codes. 

\section{Elaboration}

\subsection{The algorithm}

We define the following elaboration rules, using a bidirectional type checking syntax.

First we have the type synthesis rules, which defines judgements $\core{\Gamma} \vdash t \syn \core{t'} \in \core{T}$, where 
everything in red is core syntax.

\begin{mathpar}
  \ebrule[syn-var]{
    \hypo{\vdash \Gamma . x:S . \Delta}
    \infer1{\Gamma.x:S .\Delta \vdash x \syn x \in S}
  }
  \and
  \ebrule[syn-fun]{
    \hypo{\Gamma \vdash f \syn f' \in \Pi(S,T)}
    \hypo{\Gamma \vdash S \ni s \chk s'}
    \infer2{\Gamma \vdash f \; s \syn f' \; s' \in T[s']}
  }
\end{mathpar}

\begin{mathpar}
  \ebrule[syn-proj$0$]{
    \hypo{\Gamma \vdash p \syn p' \in \Sigma(S,T)}
    \infer1{\Gamma \vdash \pi_0 \, p \syn \pi_0 \, p' \in S}
  }
  \and
  \ebrule[syn-proj$1$]{
    \hypo{\Gamma \vdash p \syn p' \in \Sigma(S,T)}
    \infer1{\Gamma \vdash \pi_1 \, p \syn \pi_1 \, p' \in T[\pi_0\, p']}
  }
\end{mathpar}

Then, we have the type checking rules, which defines judgements $\core{\Gamma} \vdash \core{T} \ni t \syn \core{t'}$:

\begin{mathpar}
  \ebrule[chk-univ]{
    \hypo{\Gamma \vdash i+1 \leq j}
    \infer1{\Gamma \vdash \U_j \ni \U_i \chk \U_i^j }
  }
  \and
  \ebrule[chk-pi]{
    \hypo{\Gamma \vdash \U_i \ni A \chk A' }
    \hypo{\Gamma.\decode{A'}_i \vdash \U_i \ni B \chk B' }
    \infer2{\Gamma \vdash \U_i \ni \Pi(A,B) \chk \Pi^i(A',B') }
  }
\end{mathpar}

\begin{mathpar}
  \ebrule[chk-data]{
    \hypo{\Gamma \vdash \U_i \ni \overline{u} \chk \overline{u'} }
    \infer1{\Gamma \vdash \U_i \ni \D(\overline{u}) \chk \D^i(\overline{u'}) }
  }
  \and
  \ebrule[chk-sigma]{
    \hypo{\Gamma \vdash \U_i \ni A \chk A' }
    \hypo{\Gamma.\decode{A'}_i \vdash \U_i \ni B \chk B' }
    \infer2{\Gamma \vdash \U_i \ni \Sigma(A,B) \chk \Sigma^i(A',B') }
  }
\end{mathpar}

\begin{mathpar}
  \ebrule[chk-lambda]{
    \hypo{\Gamma.S \vdash t \ni t \chk t'}
    \infer1{\Gamma \vdash \Pi(S,T) \ni \lambda x.t \chk \lambda (x:S).t'}
  }
  \and
  \ebrule[chk-pair]{
    \hypo{\Gamma \vdash S \ni s \chk s'}
    \hypo{\Gamma \vdash T[x:=s'] \ni t \chk t'}
    \infer2{\Gamma \vdash \Sigma(S,T) \ni (s,t) \chk (s', t')}
  }
\end{mathpar}

We also need a mode switch rule, which relies on a \textit{coercion} judgement. 
\begin{mathpar}
  \ebrule[switch]{
    \hypo{\Gamma \vdash E \syn M \in B }
    \hypo{\Gamma \vdash \cast M : B \to A \rightsquigarrow M'}
    \infer2{\Gamma \vdash A \ni E \chk M'}
  }
\end{mathpar}

We now describe the coercion judgement $\core{\Gamma} \vdash \cast \core{M} : \core{B} \to \core{A} \rightsquigarrow \core{M'}$:

\begin{mathpar}
  \ebrule[coe-pi]{
    \hypo{\Gamma.(x:A') \vdash \cast x: A' \to A \rightsquigarrow u(x)}
    \hypo{\Gamma.(x:A') \vdash \cast M(u(x)): B \to B' \rightsquigarrow N(x)}
    \infer2{\Gamma \vdash \cast M : \Pi(A,B) \to \Pi(A',B') \rightsquigarrow \lambda x. \; N(x)}
  }
  \and
  \ebrule[coe-sigma]{
    \hypo{\Gamma.(x:A') \vdash \cast x: A' \to A \rightsquigarrow u(x)}
    \hypo{\Gamma.(x:A') \vdash \cast M(u(x)): B \to B' \rightsquigarrow N(x)}
    \infer2{\Gamma \vdash \cast M : \Sigma(A,B) \to \Sigma(A',B') \rightsquigarrow (u(x),\; N(x))}
  }
  \and
  \ebrule[coe-univ]{
    \hypo{\Gamma \vdash i \leq j}
    \infer1{\Gamma \vdash \cast M : \U_i \to \U_j \rightsquigarrow \ \uparrow_i^j M}
  }
  \and
  \ebrule[coe-unify]{
    \hypo{\Gamma \vdash A \equiv B}
    \infer1{\Gamma \vdash \cast M : A \to B \rightsquigarrow \ M}
  }
\end{mathpar}

With that we add coercions from metas to negative types, that tries to infer the strucure of the meta,
and propagate the coercion by creating new fresh metas:
\begin{mathpar}
  \ebrule[coe-meta-pi]{
    \hypo{\Gamma \vdash \cast  M : \Pi(\beta_1, \beta_2) \to \Pi(A,B) \rightsquigarrow M'}
    \infer1{\Gamma \vdash \cast M : \alpha \to \Pi(A,B) \rightsquigarrow M'}
  }
  \and
  \ebrule[coe-meta-sigma]{
    \hypo{\Gamma \vdash \cast  M : \Sigma(\beta_1, \beta_2) \to \Sigma(A,B) \rightsquigarrow M'}
    \infer1{\Gamma \vdash \cast M : \alpha \to \Sigma(A,B) \rightsquigarrow M'}
  }
  \and
  \ebrule[coe-meta-univ]{
    \hypo{\Gamma \vdash \cast  M : \U_\beta \to \U_i \rightsquigarrow M'}
    \infer1{\Gamma \vdash \cast M : \alpha \to \U_i \rightsquigarrow M'}
  }
\end{mathpar}

The idea is that we infer as much as we can on the structure of the type, and propagate the coercion. Concretely,
we extend András Kovács' algorithm on observing definitional equality, to handle casting simultaneously. This solves
the issues related to the interplay between implicit coercions and unification. Of course, the rule \textsc{coe-unify} 
should be applied only if all the other rules failed.  

\begin{exemple}
    If we try to do a coercion from a meta to a positive type, we end up calling directly the unifier. For instance,
    with an identity type: $\cast M : \alpha \to (a =_A b)$ directly produces a unification problem $\alpha \overset{?}{\equiv} (a =_A b)$.
\end{exemple}

\subsection{Soundness}

We can prove a soundness result for our bidirectional type checking algorithm, which ensures that we
are translating to well-typed terms. First, we prove that the casting judgement does produce a term of the
right type.

\begin{theoreme}[Soundness of Casting]\label{castsound}
    If $\Gamma \vdash M: B$ and $\Gamma \vdash \cast M : B \to A \rightsquigarrow M'$, then $\Gamma \vdash M':A$
\end{theoreme}

\begin{proof}
    By induction on the casting judgement.
\end{proof}

\begin{theoreme}[Soundess of Elaboration]
    We have the following:

    \begin{itemize}
        \item If $\Gamma \vdash t \syn t' \in T$, then $\Gamma \vdash T$ and $\Gamma \vdash t' : T$ i.e. a type 
            and term elaborated from synthesis is well-typed. 
        \item If $\Gamma \vdash T \ni t \chk t'$, then $\Gamma \vdash t' : T$ i.e. the term elaborated by checking
            an expression against type $T$ is indeed of type $T$.
    \end{itemize}

    This means that our elaboration procedure produces well-typed terms in our core language.
\end{theoreme}
\begin{proof}
    By mutual induction on the type synthesis and checking judgements, using theorem \ref{castsound} in
    the case of the cast mode switch.
\end{proof}

Note that this result is only a safeguard and does not guarantee that our algorithm is entirely well-behaved, and one
